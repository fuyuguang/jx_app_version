```

feat：新功能（feature）
fix：修改bug
docs：文档修改
style：代码格式修改（不影响代码运行的变动，例如分号修改），注意⚠️不是 CSS 修改
refactor：代码重构
test：测试用例修改
build: 影响项目构建或依赖项修改，如 webpack版本升级等
perf: 性能优化
revert: 撤销之前的 commit
ci: 持续集成相关文件修改
chore：其它修改（不在 上述类型中的修改）



git 提交规范 breaking change
```
